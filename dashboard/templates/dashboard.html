SHAP

SHAP is a method used to explain individual predictions made by any machine learning model.
It connects optimal credit allocation with local explanations using Shapley values,
a concept from cooperative game theory.


Core idea : Every feature value in a model's prediction is treated as a "player" in a game.
The prediction itself is the "payout." SHAP values fairly distribute the total difference
between the actual prediction (output) and the average prediction (baseline) among the features.




SHAP Additive Feature Attribution Formula

Why Use SHAP for Your XplainAI Models?

Rationale Explanation
Model Agnostic SHAP can be applied to any of your four models (e.g., Random Forest, Logistic Regression, Neural Network) without needing to change the explanation method.
Local Accuracy The sum of the Shapley values plus the base value equals the model's actual prediction. This ensures the explanation is mathematically sound and true to the model's output.
Fairness (Game Theory) SHAP values are the only method based on solid game theory principles (Shapley values) that guarantee a fair distribution of the prediction outcome among all contributing features.
Global & Local Insight SHAP can be used for both: Local Explanation (why one specific person was approved for a loan) and Global Explanation (which features, like Credit Score, are most important across all loan applications).
Addressing the Statement By using SHAP, you directly address the "black box" statement. The output for each model's prediction will be a list of features with their corresponding SHAP values, clearly visualizing why the decision was made.
